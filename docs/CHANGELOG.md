## 2025-02-10

### プロジェクト初期構築

- **ディレクトリ構成の確立**
  - `src/main.ts` をエントリーポイントとする基本構成を構築
  - `src/engine/` — シミュレーション・レンダリング・オーディオの3モジュール体制
  - `src/components/` — タイマー制御・UI の2モジュール体制
  - `src/utils/` — PRNG シード・URL パラメータのユーティリティ
  - `dist/` — ビルド出力先
  - `docs/` — ドキュメント管理

- **PRD.md の作成**
  - コンセプト: "Chaos to Order, Randomness to Truth."
  - ブザービーター方式による粒子放出アルゴリズムの定義
  - 50/50 純粋確率分岐（物理より数学を優先する設計思想）
  - Static Baking による描画最適化方針の決定
  - URL パラメータ (`?t=300&m=1&s=42`) による状態共有機能の要件定義
  - ビジュアルデザインシステム: 黒背景 + JetBrains Mono + ミニマリズム
